## Estatistica por posição
### Point Guard Stats:
    PTS : Points per game
    AST : Assists per game
    STL : Steals per game
    TOV : Turnovers per game (Quanto menor melhor)
  
### Shooting Guard Stats:
    PTS : Points per game
    3P% : 3-point field goal percentage
    STL : Steals per game
    TRB : Total rebounds per game

### Small Forward Stats:
    PTS : Points per game
    TRB : Total rebounds per game
    AST : Assists per game
    STL : Steals per game

### Point Forward Stats:
    TRB : Total rebounds per game
    PTS : Points per game
    BLK : Blocks per game
    2P% : 2-point field goal percentage

### Central Stats:
    TRB : Total rebounds per game
    BLK : Blocks per game
    PTS : Points per game
    2P% : 2-point field goal percentage
